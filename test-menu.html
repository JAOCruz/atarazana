Mobile Menu Test Page

This page is designed to test the mobile menu functionality. Try clicking on the hamburger menu icon in the top right corner (on mobile view).

Testing Instructions:

  1. Resize your browser to mobile width (less than 768px)
  2. Click on the hamburger menu icon
  3. The mobile menu overlay should appear
  4. Click on the "Eventos" dropdown
  5. The dropdown menu should expand
  6. Click the X to close the menu
